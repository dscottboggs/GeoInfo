    geoinfo

    Geocoding and reverse-geocoding from the geo-info.co API in Crystal.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     geoinfo:
   github: dscottboggs/geoinfo

    `

    1. Run shards install

    Usage

    require "geoinfo"
GeoInfo::City.new(city: "New York", state: "New York", country: "US")

    See src/city.cr for members available, and spec/city_spec.cr for more detailed usage examples.
